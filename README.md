# 💧 Water Contamination & Risk Prediction System

A **Machine Learning-based web application** that analyzes water quality parameters to predict water potability, assess contamination risk, and recommend suitable treatment measures for safer drinking water.

---

## ✨ Features

- 💧 Predicts water potability
- ⚠️ Assesses contamination risk levels
- 📊 Analyzes multiple water quality parameters
- 🤖 Machine Learning-based predictions
- 💡 Provides treatment recommendations
- 🌐 User-friendly web interface

---

## 🎯 Objectives

- Predict whether water is safe for drinking
- Assess contamination risk using water quality data
- Provide treatment suggestions based on prediction results
- Promote awareness of water quality and public health
- Support quick decision-making through automated analysis

---

## 🛠️ Tech Stack

- 🐍 Python
- 🌐 Streamlit
- 🤖 Machine Learning
- 📊 Scikit-learn
- 📈 Pandas & NumPy
- 📉 Matplotlib

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Rakshithsreddy123/Water-Contamination-Risk-Prediction.git
```

### Navigate to the Project

```bash
cd Water-Contamination-Risk-Prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

---

## 📊 Output

The application analyzes the entered water quality parameters, predicts whether the water is safe for drinking, estimates the contamination risk level, and provides appropriate treatment recommendations.
